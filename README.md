# MedRise Project

This project has been prepared for GitHub continuation and collaboration.

## Tech Stack
- Node.js
- TypeScript
- PNPM Workspace

## Project Setup

### 1. Install PNPM
```bash
npm install -g pnpm
```

### 2. Install dependencies
```bash
pnpm install
```

### 3. Run the project
```bash
pnpm run build
```

## Useful Commands

### Type checking
```bash
pnpm run typecheck
```

### Build project
```bash
pnpm run build
```

## GitHub Setup

Initialize Git:

```bash
git init
git add .
git commit -m "Initial commit"
```

Create a GitHub repository and connect it:

```bash
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git push -u origin main
```

## Notes
- This repository uses PNPM workspaces.
- Ensure you are using PNPM instead of npm or yarn.
